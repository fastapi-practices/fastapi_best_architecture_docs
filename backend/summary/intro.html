基于 FastAPI 构建的企业级后端架构解决方案

伪三层架构

mvc 架构作为常规设计模式，在 python web 中很常见，但是三层架构更令人着迷

在 python web 开发中，三层架构的概念并没有通用标准，所以这里我们称之为伪三层架构

！但请注意 ！

我们并没有传统的多 app （微服务）目录结构（django、springBoot...），而是自以为是的目录结构

如果您不喜欢这种模式，可以对其进行任意改造！
