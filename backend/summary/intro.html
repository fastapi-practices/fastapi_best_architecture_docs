基于 FastAPI 构建的企业级后端架构解决方案

三层架构

MVC 架构是 Python Web 项目中常见的设计模式。fba 采用更贴近后端工程分层的三层架构，将接口、业务和数据访问边界拆开，便于协作和维护

注意

请注意

fba 没有采用传统多应用或微服务目录结构，而是根据 FastAPI 后端工程实践组织目录，详情见项目结构

如果此结构不适合你的团队，可以在保留分层边界的前提下自行调整
