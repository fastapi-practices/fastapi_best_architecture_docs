::: caution
前端已默认集成【字典】功能，所以 fba 必须安装字典插件（已内置）并执行插件中的 SQL 脚本
:::

::: steps

1. 准备本地环境

   * Node.js（20.15.0 及以上版本）

2. 拉取 Git 仓库

   ```shell
   git clone https://github.com/fastapi-practices/fastapi_best_architecture_ui.git
   ```

3. 安装依赖

   ```shell
   pnpm install
   ```

4. 启动

   ```shell
   pnpm dev
   ```

:::
