安装

插件安装方式包含两种

1. 手动
  1. 获取插件仓库源码并下载，将源码中的插件目录直接拷贝到 fba 项目对应的目录下
  2. 如果插件包含依赖，执行 backend/scripts/init_plugin.py 文件安装依赖
  3. 根据插件使用说明完成安装
2. 自动

  1. 获取打包好的插件 zip 压缩包

    • 插件 GitHub 仓库中的 Release zip 压缩包

      示例

    • 通过插件打包接口创建的 zip 压缩包

  2. 将压缩包通过插件安装接口进行安装

  3. 根据插件使用说明完成安装
