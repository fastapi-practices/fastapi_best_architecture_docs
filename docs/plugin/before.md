## 愿景

提供一个共创平台，告别高耦合集成，让功能变得像乐高一样随意拼装

项目会提供基础插件管理能力用于安装、启停、下载和卸载。插件市场主要用于展示和导航，第三方插件的质量、更新、收费和技术支持仍由插件开发者负责
