## 函数命名

fba 遵循以下命名规范：

- 获取/查询详情： `get()`
- 通过 xxx 获取/查询详情：`get_by_xxx()`
- 获取/查询列表表达式：`get_select()`
- 获取/查询列表：`get_list()`
- 获取/查询所有：`get_all()`
- 子查询：`get_children()`
- 关系查询：`get_with_relation()`
- 创建：`create()`
- 更新：`update()`
- 删除：`delete()`
