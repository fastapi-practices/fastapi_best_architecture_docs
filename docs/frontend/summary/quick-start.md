---
title: 快速开始
---

::: warning
前端已默认集成<字典>功能，所以后端必须配置<字典插件>，并执行字典插件中的 SQL 脚本
:::

::: steps

1. 准备本地环境

    - Node.js（20.15.0 及以上版本）
    - <p>pnpm</p>

2. 拉取 Git 仓库

   ```shell
   git clone https://github.com/fastapi-practices/fastapi_best_architecture_ui.git
   ```

3. 安装依赖

   ```shell
   pnpm install
   ```

4. 启动

   ```shell
   pnpm dev
   ```

:::
